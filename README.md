# DOM Manipultaion
- What is DOM? How it represents the HTML structure?\
**Answer:** DOM stands for Document Object Model which is a tree like structure created by our browser when a HTML page loads. Every tag in the HTML page is a node of the tree.

- Types of Node in DOM Tree\
**Answer:** Element node, text node, comment node, attribute node, document node and doctype node.

- Difference between text node and element node.\
**Answer:** Element node is actual tag and can have children nodes but Text node is the content and can't have children.

- Difference between getElementById and querySelector.\
**Answer:** getElementById is used to select elements by ID while querySelector can be used to select element by id, class or element's name just like css.

- What does getElementsByClassName return? Is it an array?.\
**Answer:** This returns an array like structure called NodeList.

- When should you use textContent and when should you use innerText?\
**Answer:** textContent is faster than innerText. textContent can be used to change even the hidden content but innerText can't do that.
